## Three.js:
   En esta parte de la tarea hice el triangulo en Three.js en el cual se dibuja un triángulo con colores por vértice (rojo, verde y azul) y se ve el degradado/interpolación dentro del triángulo. El es oscuro (pero no negro). Para que funcione correctamente debes hacer lo siguiente: 

1) Abre la carpeta del proyecto en VS Code (la carpeta donde está el archivo package.json).

2) Luego abre la terminal de VS Code.

3) Entra en la subcarpeta mi-three usando el comando: cd mi-three

4) La primera vez que la uses, instala lo necesario escribiendo: npm install. 

5) Luego lo ejecutas con: npm run dev

6) En la terminal saldra un link parecido a este: http://localhost:5173/. Copia ese link y abrelo en el navegador.

7) Para saber si sí funcionó, tiene que salirte el fondo oscuro y el triángulo con el degradado de colores.

8) Para detenerlo, en la terminal presiona Ctrl + C.
